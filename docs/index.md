# KaSON documentation

[Tutorial](tutorial.md) · [API guide](api-guide.md) ·
[Compatibility](compatibility.md)

KaSON is an allocation-free C99 JSON parser. This documentation is organized
by task:

- Follow the [tutorial](tutorial.md) for complete regular, streaming, and schema
  examples.
- Read the [API guide](api-guide.md) for behavior, ownership, lifetimes, errors,
  and lower-level helpers.
- Use the generated modules and symbol index for the exact declaration of each
  public API.
- Check the [compatibility policy](compatibility.md) before upgrading or
  distributing a binary package.

The repository README contains build, installation, benchmark, and development
commands.
